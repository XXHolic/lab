居中效果需要有一个参照物，下面提到的方法前提是，都有一个一定高度和宽度的父元素。

经常遇到的居中效果主要有2种：水平居中和垂直居中。需要居中的元素高宽度可能有或没有，为了方便理解，先以有固定高宽度元素为例

水平居中

方法1：如果居中元素是块级元素，使用margin: 0 auto;

方法2：改变居中元素display属性,父元素设置text-align: center;

方法3：居中元素使用定位position和margin负值

方法4：居中元素使用定位position和transform

方法5：父元素使用flex

垂直居中

方法1：使用定位position和margin负值

方法2：使用定位position和transform

方法3：父元素使用flex

方法4：使用table的特性，这种可以让居中元素内部的元素居中，元素本身改变了属性，高宽度失效，不符合假定的场景
