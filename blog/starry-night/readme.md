## 动态结合象征点
- 星空的的 12 个点可以看做代表一天的一半
- 下面衍升的植物，可以换成动画恶之华里面的那个眼睛，没出现一个星星，就睁开一点，出现最后一个的时候，全部睁开，眼睛里面可以换成写轮眼。
- 星空可以换成 7 个，结合西方 7 天创造世界的传说，配合相关的文字说明。
